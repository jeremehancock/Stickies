How it works

  • Add — double-click the board, or hit + Note.
  • Type — click a note and start writing.
  • Move — just drag it. It lifts, tilts and settles back.
  • Resize — drag the corner grip. Notes keep their proportions; long text scrolls inside.
  • Markdown — write Markdown (headings, bold, lists, links…). It renders when you click away; click back in to edit the source. Online only.
  • Change Color — hover a note and click the color button to pick a new color.
  • Font — hover a note and pick from 5 fonts: Sans Serif, Serif, Monospace, Handwritten, or Rounded. Each note can use a different font.
  • Text Size — hover a note and choose Small, Medium, Large, or X-Large. Applies to all text in the note, including rendered Markdown.
  • Theme — toggle light & dark from the top bar; dark is the default.
  • Delete — hover a note and hit ×.
  • Undo a delete — tap Undo on the toast, or press Ctrl/⌘+Z.
  • Rename — click the board name in the top bar and type.
  • Export / Import — back up this board to a .json file, or load one back in. Importing replaces everything on this device.

Storage: notes are saved only in this browser (localStorage) — not synced, shared, or backed up. Clearing browser data or using a private window wipes them. Treat it as a scratchpad, and Export to keep a copy.

Clear the board?

This removes all notes. It can't be undone.

Export this board

Save everything on this board — every note with its text, color and position, plus the board name — into a single file you can keep.

  • Use it as a backup, or to move this board to another browser or device (open it there and Import).
  • Downloads to your device as stickies-board.json.
  • Nothing is uploaded — the file is built right here in your browser.

Import a board

Load a board from a Stickies .json file you exported earlier.

⚠ This completely replaces your current board. Every note and the board name saved in this browser will be overwritten and can't be recovered. If you want to keep what's here, Export it first.
